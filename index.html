EPCIS standard

Electronic Product Code Information Services (EPCIS) is a GS1-developed standard that enables the capture and exchange of visibility data information as an event both within and between organizations for all types of physical and digital assets. The initial version EPCIS 1.0 was released in 2007 and the most recent version EPCIS 2.0 has be released in 2022 with many enhancements.

Key features of EPCIS

Tracking

Find exactly where the items are present using the most recent EPCIS event.

Tracing

Track the objects to their origin using all EPCIS events associated with objects.

Analysing

All the EPCIS events collected over a particular time can be interpreted in a specific business context.
